# triangulation

* C++
* cmake
* SDLv2

## requirements

Ubuntu requirements 

```
$ sudo apt install cmake g++
$ sudo apt install libsdl2-dev
$ sudo apt install libsdl2-ttf
```

MacOS 

```
% brew install cmake
% brew install sdl2
% brew install sdl2_ttf
```

## Build

```
cmake -B tmp .
cmake --build tmp
```
